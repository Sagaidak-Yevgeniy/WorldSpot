Введение: что такое WorldSpot

WorldSpot — географическая игра, вдохновлённая GeoGuessr. Игрок видит панораму улицы или природы и должен указать на карте мира, где, по его мнению, сделан снимок. Чем ближе угадывание к реальной точке, тем больше очков.

Это пособие проведёт вас от пустой папки до работающего прототипа на Python и pygame и опубликованного репозитория на GitHub. Мы создаём не клон GeoGuessr, а свою игру с уникальной статистикой: процент успеха по странам и регионам вместо флагов.

Особенности WorldSpot

  • Знакомый геймплей: панорама → догадка на карте → результат.
  • Минималистичный современный интерфейс.
  • Обычный и рейтинговый режимы.
  • Подробная статистика после каждого раунда.
  • Процент угадываний по странам и регионам (без флагов).
  • История игр, достижения и личные рекорды.

Что вы создадите

  • Окно игры с панорамой и интерактивной картой мира.
  • Систему очков по расстоянию (формула Haversine).
  • Экран результатов с точностью в км и очками.
  • Таблицу статистики: % по странам, лучшие и худшие регионы.
  • Сохранение истории и достижений в JSON.
Практика: Старт
  1. Установите Python 3.10+ и создайте папку WorldSpot.
  2. Откройте проект в VS Code или Cursor.
  3. Прочитайте README.md в корне шаблона.

Анатомия игры в стиле GeoGuessr

Классический раунд GeoGuessr состоит из пяти этапов: показ локации, исследование (поворот камеры, движение — в полной версии), выбор точки на карте, подсчёт очков, показ правильного ответа. WorldSpot реализует ядро этого цикла.

Игровой цикл раунда

1. PANORAMA — игрок изучает изображение локации.

2. GUESS — клик по карте мира, подтверждение.

3. REVEAL — линия от догадки до правды, расстояние, очки.

4. STATS — обновление % по стране и региону.

5. NEXT — следующая локация или итоги серии.

def pick_round(locations, count=5):
    pool = locations.copy()

Игровой цикл и сцены

# src/main.py
import pygame
from game import Game
from settings import WIDTH, HEIGHT, FPS, TITLE

def main():
    pygame.init()
    screen = pygame.display.set_mode((WIDTH, HEIGHT))
    pygame.display.set_caption(TITLE)
    clock = pygame.time.Clock()
    game = Game(screen)
    running = True
    while running:
        dt = clock.tick(FPS) / 1000.0
        for e in pygame.event.get():
            if e.type == pygame.QUIT: running = False
            game.handle_event(e)
        game.update(dt)
        game.draw()
        pygame.display.flip()
    pygame.quit()

Game переключает сцены: MenuScene → RoundScene → ResultsScene → StatsScene. Каждая сцена знает, как рисовать UI и обрабатывать клики.

Карта мира и выбор точки

Карта — прямоугольное изображение мира (экваториальная проекция). Клик (mx, my) внутри MAP_RECT переводится в широту и долготу.

def pixel_to_latlon(mx, my, map_rect):
    x = (mx - map_rect.x) / map_rect.w
    y = (my - map_rect.y) / map_rect.h
  lon = x * 360.0 - 180.0
  lat = 90.0 - y * 180.0
  return lat, lon

def latlon_to_pixel(lat, lon, map_rect):
    x = (lon + 180.0) / 360.0
    y = (90.0 - lat) / 180.0
    px = map_rect.x + x * map_rect.w
    py = map_rect.y + y * map_rect.h
    return int(px), int(py)

После клика покажите маркер (красный круг). Кнопка «Угадать» фиксирует выбор.

Практика: Карта
  1. Нарисуйте карту мира 500×250 или используйте assets/ui/world_map.png.
  2. По клику ставьте маркер и выводите lat/lon в углу экрана.
  3. Заблокируйте повторный клик после подтверждения.

Система очков: формула Haversine

GeoGuessr начисляет до 5000 очков. Чем меньше расстояние между догадкой и правдой, тем выше счёт. Расстояние считают по сфере Земли — формула Haversine.

import math

def haversine_km(lat1, lon1, lat2, lon2):
    R = 6371.0
    p = math.pi / 180
    a = (math.sin((lat2-lat1)*p/2)**2 +
         math.cos(lat1*p)*math.cos(lat2*p)*math.sin((lon2-lon1)*p/2)**2)
    return 2 * R * math.asin(math.sqrt(a))

def score_from_distance(km, max_score=5000):
    if km <= 0.05: return max_score
    return max(0, int(max_score * math.exp(-km / 2000)))

Подберите делитель (2000) под желаемую сложность: меньше — щедрее очки.

Статистика по странам и регионам

Вместо флагов WorldSpot показывает процент успешных угадываний. Успех — если расстояние меньше порога (например, 500 км) или очки выше 4000.

# src/systems/stats.py
class RegionStats:
    def __init__(self):
        self.by_country = {}  # страна -> {correct, total, avg_km}

    def record(self, country, region, distance_km, score):
        c = self.by_country.setdefault(country, {'ok':0,'n':0,'km':[]})
        c['n'] += 1
        c['km'].append(distance_km)
        if distance_km < 500 or score >= 4000:
            c['ok'] += 1

    def percent(self, country):
        c = self.by_country[country]
        return 0 if c['n']==0 else round(100*c['ok']/c['n'])

Что показывать игроку

  • Процент угадываний по каждой стране.
  • Средняя точность (среднее расстояние в км).
  • Лучшие регионы (высокий %) и худшие (низкий %).
  • График прогресса по датам (из history.json).

Режимы: обычный и рейтинговый

Обычный режим — без таймера, можно изучать панораму сколько угодно. Рейтинговый — лимит времени на раунд (3 минуты), бонус за скорость.

class RoundScene:
    def __init__(self, game, location, ranked=False):
        self.ranked = ranked
        self.time_left = ROUND_TIME_RANKED if ranked else None

    def update(self, dt):
        if self.ranked and self.time_left is not None:
            self.time_left -= dt
            if self.time_left <= 0:
                self.auto_submit_worst_guess()

Полный цикл раунда в коде

1. MenuScene: выбор «Играть» / «Рейтинг» / «Статистика».

2. RoundScene: загрузка location, показ панорамы и карты.

3. Игрок кликает карту → нажимает «Угадать».

4. scoring.haversine_km + score_from_distance.

5. stats.record(country, region, km, score).

6. ResultsScene → кнопка «Далее» или итог серии.
